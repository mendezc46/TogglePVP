# Toggle PVP

Mod servidor para Minecraft 1.19.2 Forge que permite activar/desactivar PVP.

## Comandos

- `/pvp on` - Activar PVP
- `/pvp off` - Desactivar PVP
- `/pvpstats` - Ver kills y deaths
- `/pvpall [free|locked-on|locked-off]` - Control global (solo OPs)

## Características

- PVP desactivado por defecto
- Combat lockdown de 60 segundos
- Cooldown de 5 segundos entre toggles
- Mostrar timer en actionbar
- Mata al jugador si se desconecta durante combate
- Detecta crash del servidor
- Estadísticas de kills/deaths persistentes
- Modo global: libre, forzado ON, forzado OFF

## Instalación

Copiar `togglepvp-1.0.0.jar` en la carpeta `mods/` del servidor.
